# 🌟 24/7 GPT‑4 Streamlit Chatbot

An all-in-one Streamlit app featuring:
- **GPT‑4-powered Q&A** (general + dataset-focused)
- **CSV upload → dynamic visualizations**
- **Forecasting** via Prophet & Chronos
- **Session memory** for chat context
- **Location-based tools**: live weather & AI imagery

## Setup
- Upload to GitHub
- Deploy via Streamlit Cloud
- Set secret: `OPENAI_API_KEY = "your-key"`

Enjoy 🎉
